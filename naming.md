# Naming Conventions

Street names mapped to brew types for the Dalston Rooftop Brewery.

| Street Name | Brew Type |
|-------------|-----------|
| De Beauvoir | Saison |
| Ridley Road | Spicy Mead |
| Junction | Basic Mead |
| Wilton Way | Citrus Mead |
| Deorlaf's Tun | Pale Ale (Voss yeast) |
| Fenton Close | Ginger Ale (alcohol free) |
| Colvestone | Berry Mead |
| Navarino Road | Hibiscus Mead |

## Usage

Batch names follow the pattern: `{Street Name} #{Batch Number}`. The only exception is the first batch, which doesn't include the batch number.

Examples:
- Junction (first batch of Basic Mead)
- Wilton Way #2 (second batch of Citrus Mead)
- De Beauvoir #3 (third batch of Saison)
